Mobile Haptics - Property Controls

Mode
Preset: use a built-in haptic. Custom: use a pattern from https://haptics.lochie.me/ (paste in Value).

Preset
Shown when Mode is Preset. Built-in pattern: Success, Nudge, Error, Buzz, Selection, Light, Medium, Heavy.

Value
Shown when Mode is Custom. Create your custom haptic feedback at haptics.lochie.me and paste the trigger([...]) pattern.

On Desktop
When haptics are not available (e.g. desktop): Silent or Play Sound. Default: Silent.

Usage
Place Mobile Haptics inside a frame. The component listens for taps/clicks on that frame’s parent (grandparent of this component). When that parent is tapped (e.g. on mobile), the chosen haptic runs. More components at Framer University.
