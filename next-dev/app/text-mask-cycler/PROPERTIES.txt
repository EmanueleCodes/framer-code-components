Text Mask Cycler - Property Controls

Preview
In the Framer canvas, On runs the full cycle. Off shows the first word only with the mask hidden so the layout stays readable.

Offscreen
When Pause, the cycle stops while the layer is outside the viewport. When Run, it keeps animating off screen.

Words
List of word and blind-color pairs used by the cycle.

Color
Single text color used for all words.

Font
Typography style used for the visible word.

Delay
Time each word stays visible before the blinds transition.

Direction
Direction used for the blinds movement.

In
Transition settings for the blind cover phase.

Out
Transition settings for the blind uncover phase. More components at [Framer University](https://frameruni.link/cc).
