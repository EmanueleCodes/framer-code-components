Circle Animator - Property Controls

Preview
Shows or hides the animation while editing in Framer. Turn it off to position elements without distraction.

Mode
Choose between displaying Images (upload your own) or Components (connect Framer components).

Content
When using Components mode, this is where you add the Framer components you want to animate in a circle.

Count
Sets how many items appear in the circle (only visible in Images mode). Choose between 2 and 10 items.

Sizing
Fixed: All components are sized to the same width and height.
Fit content: Each component keeps its natural size.

Width & Height
Sets the size of each item in pixels. Hidden when using fit-content sizing with components.

Image 1-10
Upload slots for your images. The number of visible slots adjusts based on your Count setting.

Radius
Controls how far the items are from the center. Larger values create a bigger circle.

Speed
Adjusts how fast items rotate around the circle. Lower values = slower, higher values = faster.

Orientation
Fixed: Items rotate as they move around the circle.
Pin: Items stay upright (like a Ferris wheel).

Rotation
(Only visible when Orientation is set to Fixed)
Fixed angle: All items rotate to a specific angle you set
Radial: Items point outward from the center
Tangent: Items align with the circular path direction

Angle
(Only visible with Fixed angle rotation)
Set the exact rotation angle for your items in degrees.

Direction
Choose whether items move Clockwise or Counter-clockwise around the circle.

On Hover
Set to Pause to stop the animation when someone hovers their mouse over it. Set to None to keep it running.
