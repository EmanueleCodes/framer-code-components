CobeGlobe - Property Controls

Preview
Runs the globe animation in the Framer editor canvas. On / Off.

Motion
Phi (horizontal rotation), Tilt (theta), Spin auto-rotate, Speed when Spin is on, Drag for pointer-based spring rotation.

Globe
Theme (Light / Dark), Dark blend when Light, Diffuse, Samples (mapSamples), Land (mapBrightness), Ocean (mapBaseBrightness), Scale, Opacity, OffsetX / OffsetY, Sharp (device pixel ratio).

Dots
Size (marker size), Lift (markerElevation), Points (lat/lng + Id per marker).

Arcs
Show, Width (arcWidth), Height (arcHeight), Links (From / To marker Ids).

Colors
Base, Marker, Glow, Arc.

Background
Optional color behind the globe; leave unset for transparent.

Debug
More components at [Framer University](https://frameruni.link/cc).
