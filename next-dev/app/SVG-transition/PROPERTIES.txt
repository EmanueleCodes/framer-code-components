SVG Transition - Property Controls

Type
Input method for the SVG. File or Code.

SVG
Upload an SVG file (when Type is File).

Code
Paste SVG code directly (when Type is Code).

Color
Stroke color.

Start
Trim along the path: start of the visible segment (0–100). Bind to variants; the component eases toward new values over Duration (Framer often snaps stroke props).

End
Trim along the path: end of the visible segment (0–100).

Width
Stroke width in pixels (single value). Eased with Start/End when it changes.

Duration
Seconds to ease Start, End, and Width when they change (0 = instant). Ease-out cubic. On the Framer canvas, values snap so the editor stays light.

Cap
Line cap. Round, Butt, or Square.

Join
Line join. Round, Bevel, or Miter.

Debug
When On, logs draw snapshots to the console. If two different instance IDs alternate or values flicker A/B, check for duplicate layers or two components on the same frame. More components at [Framer University](https://frameruni.link/cc).
