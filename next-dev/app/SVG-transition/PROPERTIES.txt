SVG Transition - Property Controls

Type
Input method for the SVG. File or Code.

SVG
Upload an SVG file (when Type is File).

Code
Paste SVG code directly (when Type is Code).

Color
Stroke color for the path animation.

Start
Starting stroke width in pixels.

End
Ending stroke width in pixels. Animates from Start to End during the transition on the published site.

Cap
Line cap style. Round, Butt, or Square.

Join
Line join style. Round, Bevel, or Miter.

Preview
Static preview in the canvas (50% trim). Dynamic runs the transition.

Mode
Animation style. Forward draws the path (visible grows from path end). Backward erases the path (visible shrinks toward path end). Forward from start draws from the path's start point. Backward from start erases toward the path's start point. Hidden when Preview is Static.

Transition
Timing configuration for the animation. More components at [Framer University](https://frameruni.link/cc).

Canvas
In the Framer canvas, stroke thickness uses Start or End depending on which control you edited most recently (not a blend between them).
