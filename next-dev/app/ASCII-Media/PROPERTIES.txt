ASCII - Property Controls

Preview
When On, animated effects (Matrix style, noise, wave, glitch) run in the Framer canvas, and **Video** with **Autoplay** advances frames in the canvas. When Off, those pause in the canvas to save performance.

Source
**Image** or **Video**. Switches which media inputs are used and which empty-state message appears.

Image
Responsive image for the ASCII effect. Hidden when **Source** is **Video**.

Video
**URL** vs **Upload** for the video file. Shown only when **Source** is **Video**.

File
Framer file upload (mp4 / webm). Shown when **Source** is **Video** and **Video** is **Upload**.

URL
Direct link to an mp4 or webm. Shown when **Source** is **Video** and **Video** is **URL**.

Poster
Optional responsive image used **only when Source is Video and Autoplay is Off**. If set, it replaces the video’s first frame as the still used for ASCII. If empty, the component seeks to **t=0** and uses the **first decoded video frame** as the poster.

Autoplay
When On, the video tries to play (muted, inline) so the canvas can sample frames. In the Framer canvas, this runs when **Preview** is also On (same idea as **ASCII Media**). When Off, the effect samples a **still** (custom **Poster** image if provided, otherwise the **first frame** at time 0). Shown only for **Video**.

Loop
Whether the video loops when playing. Shown only for **Video**.

Backdrop
Background color behind the effect. When **Underlay** is Off, the canvas is filled with this color before glyphs. When Underlay is On, transparent canvas areas fall through to the blurred photo, then this color only where nothing else draws (e.g. letterboxing). Supports **Framer color tokens** (`var(--…, #fallback)`), **`{ value: … }` wrappers**, and **light / dark pairs** (`{ light, dark }` or `Light` / `Dark`): the active branch follows Framer / OS theme (`data-framer-theme`, `color-scheme`, `prefers-color-scheme`). Values become concrete `rgba()` for canvas (see **`how-to-build-framer-components/colorInFramerCanvas.md`**), not raw `var()` on `fillStyle`.

Underlay
When On, a **second canvas** behind the glyphs is filled **per cell** with the same processed color as the ASCII layer (same sampling, **Wave**, **Curvature**, **Aberration**, **Noise**, **Bright** / **Contrast**, **Palette**, **Color**, **Scanlines**, **Vignette**, and row **Glitch** shift). Then **Blur** and **Opacity** are applied to that layer (CSS). The front canvas stays transparent except for the characters so the underlay shows through gaps.

Blur
CSS `blur()` radius in pixels for the underlay image. Only shown when Underlay is On.

Opacity
Underlay image alpha (0–1). Only shown when Underlay is On.

Font
Framer **Font** control (family, weight, italic, etc.). Drives the canvas glyph style via `fontFamily` / `fontWeight` / `fontStyle`. **Cells** still sets the raster cell size; glyph **pixel size** is derived from the cell (Framer’s font size on this control is not used for layout so the grid stays even).

Shader (object)
Groups the core ASCII look controls.

Cells
Size of each ASCII cell in pixels. Smaller values create finer detail, larger values create blockier output.

Style
ASCII preset from **`ascii_effect_presets.ts`**: Standard, Dense, Minimal, Blocks, Braille, Technical, Matrix, Hatching. Curved/wavy UVs stay clamped so cells are not skipped.

Color
Color: Preserves image colors with character patterns. Gray: Grayscale luminance only.

Bright
Additive brightness offset. Positive values lift shadows and midtones, negative darkens.

Contrast
Scales color values around mid-gray. Below 1 flattens, above 1 increases punch.

Low
Luminance threshold (0–1) used with **High** after **Bright** / **Contrast** / **Noise**. When **Low** ≤ **High**, glyphs (and matching underlay cells) only draw where luminance lies in that inclusive band. When **Low** > **High**, behavior inverts: glyphs draw only in the shadows (≤ **High**) and highlights (≥ **Low**), not in the mid-range between the two values.

High
Upper luminance bound paired with **Low**; see **Low**.

Palette
Color tinting: Original, Green, Amber, Cyan, Blue.

Effects (object)
Groups post-processing and distortion controls.

Enable
Master switch for the whole **Effects** group. **Off** hides every other control in this object and turns off all effect processing at runtime (fish-eye, aberration, scanlines, vignette, noise, wave, glitch). **On** restores the full panel and applies each sub-toggle / slider as configured.

Scanlines
Master switch (**On** / **Off**). When **On**, **Depth** and **Lines** appear; CRT scanlines run when **Depth** > 0.

Depth
Scanline band strength when **Scanlines** is On.

Lines
Number of scanline bands when **Scanlines** is On.

Vignette
Master switch (**On** / **Off**). When **On**, **Dim** and **Radius** appear; edge darkening runs when **Dim** > 0.

Dim
Vignette / edge darkening strength when **Vignette** is On.

Radius
Vignette falloff when **Vignette** is On.

Fish Eye
Barrel distortion (curvature) when **Enable** is On.

Aberration
Chromatic aberration (RGB color fringing) strength.

Noise
Master switch (**On** / **Off**). When **On**, the **Level**, **Scale**, **Frequency**, and **Speed** controls appear and grain can be applied (still needs **Level** > 0 to see it).

Level
Noise / grain strength when **Noise** is On. Animated when Level > 0.

Scale
Multiplies UV into the noise field when **Noise** is On.

Frequency
Spatial grain fineness when **Noise** is On.

Speed
Noise pattern motion over time when **Noise** is On.

Wave
Master switch (**On** / **Off**). When **On**, **Amp**, **Ripple**, and **Flow** appear and wave distortion can run (still needs **Amp** > 0 to see it).

Amp
Wave distortion strength when **Wave** is On. Animated when Amp > 0.

Ripple
Wave spatial frequency when **Wave** is On.

Flow
Wave animation speed when **Wave** is On.

Glitch
Master switch (**On** / **Off**). When **On**, **Jitter** and **Rate** appear; row shifts run when **Jitter** > 0 and **Rate** > 0. Animated when active.

Jitter
How often rows jump horizontally when **Glitch** is On.

Rate
How fast the glitch pattern advances when **Glitch** is On.

The **Effects** property group carries the Framer University link in its panel description: https://frameruni.link/cc
