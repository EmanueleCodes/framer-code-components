Noisy Ripple Reveal - Property Controls

Image
Source image to reveal.

Preview
Whether the effect runs in the Framer canvas. Static / Dynamic.

Masked
Whether the image starts masked (reveal on uncheck). When Off, only the image is visible with a transparent background.

Transition
Duration, easing, and delay for the reveal animation (same as other Framer transition controls). More components at [Framer University](https://frameruni.link/cc).

Background
Optional fill behind the reveal.

Mode
Shader source: Preset (built-in looks) or Custom (manual Shader object).

Preset
When Mode is Preset: Noisy, Sharp, Water, Water drop, or Magic. Each maps to Octaves, Warp, Freq, Liquid, Chromatic, Intensity, and Edge.

Shader
When Mode is Custom, fine-tune shader parameters. Sub-controls: Octaves, Warp, Freq, Liquid, Chromatic, Intensity, Edge.
