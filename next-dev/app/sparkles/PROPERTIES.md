Sparkles - Property Controls

Preview
When enabled, animates sparkles in the Framer canvas. When disabled, sparkles remain static but visible.

Density
Controls how many sparkles appear on screen. Higher values create more particles.

Min Size
Sets the smallest possible size for sparkles, measured in pixels.

Max Size
Sets the largest possible size for sparkles, measured in pixels.

Flicker
Controls how quickly sparkles fade in and out. Higher values create faster blinking and flickering effects.

Speed
Controls how fast sparkles move across the canvas. At 0, sparkles remain stationary. Higher values create faster movement.

Sparkle
Sets the color of the sparkle particles.

Background
Sets the background color of the component.
