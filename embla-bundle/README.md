# Embla Carousel Bundle for Framer

This bundle packages Embla Carousel and its plugins for use in Framer code components.

## What's Included

- `embla-carousel` - Core carousel functionality
- `embla-carousel-react` - React hook (useEmblaCarousel)
- `embla-carousel-autoplay` - Autoplay plugin

## Exports

```javascript
import { 
  EmblaOptionsType,
  EmblaCarouselType,
  useEmblaCarousel,
  Autoplay
} from "https://raw.githubusercontent.com/YOUR-USERNAME/YOUR-REPO/main/embla-bundle/dist/bundle.js"
```

## Build

To rebuild the bundle:

```bash
npm run build
```

This generates `dist/bundle.js` which can be imported in Framer.

## Version

- Embla Carousel: 8.6.0
- React: 18.2.0
